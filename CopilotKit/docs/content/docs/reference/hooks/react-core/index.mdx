# React Core

Documentation for the React Core package in CopilotKit.

## Key Exports

- `useCopilotChat`: Hook for interacting with the chat functionality
- `useCopilotReadable`: Hook for providing knowledge to your copilot
- `useCoAgent`: Hook for creating agents with bidirectional state sharing
